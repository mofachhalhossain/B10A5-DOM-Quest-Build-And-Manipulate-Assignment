What is the DOM in JavaScript?

The DOM (Document Object Model) is a tree-like structure representing a webpage. JavaScript uses it to access and change HTML and CSS dynamically.

What is a DOM node?

A DOM node is a part of the document, such as an element, attribute (class), or text.

How do I change content in the DOM?

Use innerHTML or textContent to change or update the content of an element.

How do I add elements to the DOM?

To add elements, use createElement() to create the element, then appendChild() or insertBefore() to add it to the DOM.
